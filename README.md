# 📘 Study Planner with GPA Goal

A **web-based study planner** that helps students plan their study schedule based on **current GPA, target GPA, and subject priorities**. This tool also generates a **colorful weekly timetable** and a **priority chart** to visualize which subjects need more focus.

---

## 💡 Features

- Add multiple subjects with:
  - Credit hours
  - Confidence level (1-5)
- Calculate subject **priority** based on credit and confidence
- Generate **focus plan** with High, Medium, Low priority indicators
- Display **priority bar chart** using Chart.js
- Generate **weekly timetable** (8AM–6PM, Mon–Fri) with:
  - Color-coded blocks based on priority
  - High-priority subjects appear more frequently
- Save subjects and priorities in **localStorage**

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (with Flexbox & Grid for timetable)
- JavaScript (ES6)
- Chart.js for bar chart visualization
- LocalStorage for saving subjects
